My 3D App Deeper Understanding

You must list here your Deeper Understanding components of your 3D App

List here those components of your 3D App module where you feel you should score 'Deeper Understanding' marks according to the specified rubric. Some examples are given, you may list more than 4, but a maximum 4 x 5 Marks will be awarded.

  • Fridge Model: The coca cola fridge model has been created in blender. The vertices in the door were seperated and then rotated to create the door effect. I then used the solidify modifier to give the fridge a thicker skin. Blenders in house meshes were used to create the shelves, plugs, door handles etc. Then further smaller cans were imported to go inside the fridge.

  • Enhanced gallery feature: The gallery is created using a mixture of Lightbox and AJAX. This allows for more functionality and creates a visually appealling end product. The thumbnails are made to enlarge when the mouse is hovered on them. The gallery grows larger/smaller dependent on screen size.

  • Blender: All the models in this webpage have been created within blender and then exported as an X3d file. Wireframe models used the Blender wireframe modifier.

  • Complex Animations: Further animations have been added such as orbit which makes the object orbit the camera and bounce which lifts the object up and then down. Using the Javascript math function I was able to make this look realistic, slowing the object down as it reached peak height.

  • Carousel feature: carousel feature created using free attribution images (links are included withing About-> Bibiliography)
