Please find some notes to help review my work easier

Thank you for marking my work. I am going to discuss a few features of the website and how I complerted them and state where I believe that I have gone above and beyond. My implementation is very different to that of the lab code. All the 3D models were created within Blender and then displayed on the site using X3D. Where the models were created in Blender, X3D has issues getting the wireframes from the models. To overcome this I used the wireframe modifier in Blender and uploaded seperate Wireframe models.

Content is also loaded dynamically in a different way to the labs, using Ajax and JSON. This can be seen in the models.php file. The 3D gallery has been produced using Ajax and Lightbox instead of using bootstrap which allowed for better visuals. I have then also used png thumbnails which get larger as you hover on them.

MVC is done differently to the labs by using Javascript to seperate my controller, view and models. This can be seen in the Model,controller and view folders.

Everything on the webpage should be working correctly. I have tested all the buttons and all the models are working as they should.

Above and Beyond
Feature Description
Fridge Model The coca cola fridge model has been created in blender. The vertices in the door were seperated and then rotated to create the door effect. I then used the solidify modifier to give the fridge a thicker skin. Blenders in house meshes were used to create the shelves, plugs, door handles etc. Then further smaller cans were imported to go inside the fridge.
Enhanced gallery feature The gallery is created using a mixture of Lightbox and AJAX. This allows for more functionality and creates a visually appealling end product. The thumbnails are made to enlarge when the mouse is hovered on them.
Complex Animations Further animations have been added such as orbit which makes the object orbit the camera and bounce which lifts the object up and then down. Using the Javascript math function I was able to make this look realistic, slowing the object down as it reached peak height.
Blender All the models in this webpage have been created within blender and then exported as an X3d file.
